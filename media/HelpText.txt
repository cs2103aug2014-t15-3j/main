Commands:

;add - Add tasks
;view - View Tasks
;delete - Delete Tasks
;edit - Edit Tasks
;undo - Undo last action
;remind - How to add a reminder
;save - How to save manually
;label - How to work on labels


If you want an in-depth help, 
simply type <help XXX> where 
XXX is any of the above 
functions.

e.g. help add
