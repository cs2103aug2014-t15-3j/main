A task can be edited in numerous ways. Following are the commands options available to edit tasks.

;edit		e.g. ;edit X ;name NewName ;i NewDescription	
;update
;change
;~
where X is the task ID associated with that task (visible in view)

You can change individual detail/property. e.g. 
;~ X ;name NewName
;~ X ;i NewDescription
;~ X ;label NewLabel
;edit X ;date
